Maven Cucumber/ Selenium Quickstart

Tool Dependencies:
   Java/ JDK - http://www.oracle.com/technetwork/java/javase/downloads/index.html
   Maven - https://maven.apache.org/download.cgi#
   ChromeDriver - https://github.com/SeleniumHQ/selenium/wiki/ChromeDriver, install in /opt/chromedriver-2.33 (c:\opt\chromedriver-2.33 for windows), rename chromedriver.exe to chromedriver for windows
 
To run test: mvn test

To clean: mvn clean
   build: mvn compile
   test: mvn test
   install: mvn install
